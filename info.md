# HackUSU 2026 — Sandbox Track Briefing Notes

## Sandbox Rules
- You may build on existing work
- Evaluated only on what you accomplish from right now
- Specify in your submission if it is not new work
- Sandbox applicants must have a team registered in the portal

## Submission (Saturday)
- **1:00 PM** — Deadline (hard, no exceptions)
- Submit via Google Form:
  - Project title & description
  - Team member names
  - Category
  - Short abstract or explanation of project
- Required for all Sandbox applicants to submit

## Judging
- **2:00 PM** — Pitch Demo (meet in Sandbox room for pitches/demos)
- **90 seconds** — hard limit
  - Business idea
  - Demo of product built

## Other Key Times
- **1:00 PM** — Project Showcase (optional)
- **4:30 PM** — Awards and Send Off
- **7:00 PM** — Deadline for Sandbox applicants to submit Application

## Success Strategies (from organizer)
1. Build a Real Company
2. Build a Functional Prototype
3. Think Deeply About Your Pitch
4. Don't Over-Engineer
5. Don't Do Surveys
6. Be Honest — Failure is OK if you learn!
